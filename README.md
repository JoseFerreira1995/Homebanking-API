# Homebanking-API

This is a Homebankink api made with HapiJs (https://hapi.dev/) and Angular(https://angular.io/)

To start: 

1: Git clone  

2: in /BackEnd directory use node index.js 

3: in /FronEnd/HomeBank-front-end directory use ng serve
